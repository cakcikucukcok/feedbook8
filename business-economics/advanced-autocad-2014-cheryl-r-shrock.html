Totally updated for AutoCAD 2014 and 2014 LT, it offers several new features as well as improved Command Line Options including, AutoComplete, and faster searching for help; Faster Start-up times; Enhanced Model and Layout tab previews. All exercises print easily on a standard 8 ½” x 11” printer. For use with the PC version of AutoCAD 2014 only.
